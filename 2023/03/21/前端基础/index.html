前端基础

学习VUE

Vue 是一套用于构建用户界面的渐进式框架


渐进式:这些功能相互独立，但可以在核心功能(轻量视图)的基础上任意选用其他的部件。不做职责之外的事。

响应式:数据改变,视图也会随之更新。会自动跟踪JS状态变化。
基于JS框架,ES语法，声明式

引入Vue3

模板

语法糖

v-once 数据改变不会渲染

v-text相当于文本插值{{msg}}，动态内容

v-html 解析html元素

v-pre跳过编译，显示原文

v-bind绑定动态标签的属性值，绑定class可以是对象、数组，（数组对象 ）也可以有多个键值对，可直接绑定对象。简写:

v-on:click 监听事件，可以绑定变量、表达式、对象；参数传递$event。简写@

v-if、v-else-if、v-else条件渲染，false时不会在DOM渲染

v-show会在DOM渲染，通过CSS的display属性进行切换

v-for 从服务器拿到数据(来自data或prop)，遍历列表，对象，数字，格式：（item,index）in 数组

this 关键字

箭头函数中使用this，会绑定到父级作用域window，而this不会按照期望指向组件实例。this绑定的是组件实例上面对应的代理事件proxy。

细节

Vnode是虚拟节点(Vue中的组件、元素)，本质是一个JavaScript的对象 template—>VDOM—>真实DOM
computed计算属性有缓存，

template元素可以当做不可见的包裹元素

定制VUE页面

  1. 找到合适的组件（加页面需要改路由）
  2. 删减不需要的元素
  3. 仿写API（输入、输出）
  4. 在组件里调用API

Vuex 是一个专为Vue应用程序开发的状态管理模式,方便组件间的数据共享

个人理解:多个组件间共享状态(变量)的插件

小范围内组件间共享数据的方式:

  • 父向子传值：v-bind属性绑定
  • 子向父传值：v-on事件绑定
  • 兄弟组件之间共享数据：EventBus
  • 接收数据的组件 $on
  • 发送数据的组件 $emit

使用详见

ElementUI 是一套为开发者准备的开源组件库，旨在快速搭建页面。

Avue 是基于现有的element-ui库进行的二次封装,组件库针对表格表单场景

学习JavaScript

详见

基础类型：

  • Number（数字）
  • String（字符串）
  • Boolean（布尔）
  • Symbol（符号）（ES2015 新增）
  • Object（对象）
    • Function（函数）
    • Array（数组）
    • Date（日期）
    • RegExp（正则表达式）
  • null（空）
  • undefined（未定义）

声明变量let，const，var

  • let 声明一个块级作用域的本地变量

  • const 声明一个不可变的常量

  • var 是最常见的声明变量的关键字

==与===区别：

  • == 忽略类型，两者相同

  • === 不忽略类型，两者相同

...x表示剩余参数
